A. General Settings
1. Enable Hotkey for switching the Addon ON / OFF
2. Enable Hotkey for adding the active Tab URL to the whitelist table
3. Enable Hotkey for removing the active Tab URL from the whitelist table
4. Open support page when the addon is upgraded
5. When a webpage has black background color, do NOT apply night mode (default is active)
6. Enable Night Mode for Top domain + all Frames (iframes) within the page (default is Top domain only)
7. Force dark body (dark gray) on all websites. Note, this option is still in Beta and may not work properly on all websites (default is not active)
8. Apply night mode for all elements with css "background-image" property. Note, this option is still in Beta and may not work properly on all websites (default is not active)
8. Apply night mode for all elements with css "background-color" property. Note, this option is still in Beta and may not work properly on all websites (default is not active)
